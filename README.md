## Requirements

- nodejs >= 12.18.1
- yarn >= 1.22.4

## Running

```bash
# Install packages
yarn install
# Start dev server locally
yarn start
```

Application will be opened automatically on [localhost:3000](http://localhost:3000).

## Testing

```bash
# run teste
yarn test
# test coverage
yarn test:coverage
```

## Build

```bash
# Build for production
yarn build
```
